# Extension Board Hardware

This directory contains the KiCad source files for the RK3588 PCB inspection extension board.

Files:

- `extension.kicad_sch`: schematic source.
- `extension.kicad_pcb`: PCB layout source.
- `extension.kicad_pro`: KiCad project file.
- `extension.csv`: exported component list.

Generated manufacturing outputs, editor history, local KiCad session files, and archived Gerber packages are not committed.
